# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.0.x   | Yes       |
| < 1.0.0 | No        |

## Reporting a Vulnerability

Please do not open a public issue for security-sensitive problems.

Report vulnerabilities by contacting the maintainer privately and include:
- affected version
- desktop environment
- session type (`x11` / `wayland`)
- reproduction steps
- expected behavior
- actual behavior
- logs or screenshots if relevant

I will acknowledge the report, investigate it, and publish a fix if the issue is confirmed.
